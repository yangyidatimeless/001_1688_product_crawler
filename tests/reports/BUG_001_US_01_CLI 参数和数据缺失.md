# US_01 BUG 文档

## Bug 详情

### BUG_001 - CLI 参数未正确定义
- **严重等级：** 高
- **复现步骤：**
  1. 进入 backend 目录
  2. 运行 `python crawler.py --help`
  3. 查看输出
- **期望结果：** 显示 --keyword、--limit、--preview、--output 等参数说明
- **实际结果：** 未显示任何参数定义
- **影响范围：** 命令行接口无法使用
- **修复建议：** 使用 argparse 库正确定义 CLI 参数

### BUG_002 - 缺少测试数据生成
- **严重等级：** 高
- **复现步骤：**
  1. 运行爬虫脚本
  2. 检查 data 目录
- **期望结果：** data 目录下生成 products.db 和 JSONL 文件
- **实际结果：** data 目录不存在
- **影响范围：** 数据存储功能无法验证
- **修复建议：** 确保爬虫运行时创建 data 目录并保存数据

---

### BUG_003 - SQLite 数据库未创建
- **严重等级：** 高
- **复现步骤：**
  1. 运行爬虫脚本
  2. 检查 products.db 文件
- **期望结果：** products.db 文件存在且包含 products 表
- **实际结果：** 文件不存在
- **影响范围：** 数据持久化功能失效
- **修复建议：** 确保爬虫初始化时创建数据库和表结构

---

### BUG_004 - 缺少网络重试机制
- **严重等级：** 中
- **复现步骤：**
  1. 检查 crawler.py 代码
  2. 搜索 retry 或重试相关逻辑
- **期望结果：** 网络请求失败时自动重试
- **实际结果：** 无重试逻辑
- **影响范围：** 网络波动时爬取可能失败
- **修复建议：** 添加重试机制（建议使用 tenacity 或 retry 库）
