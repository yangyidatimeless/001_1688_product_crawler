## 📝 使用说明

### 导出 CSV 数据

```bash
cd /app/dev_project/001_1688_product_crawler/backend

# 基础导出（使用默认关键词）
python3 crawler.py --keyword "拖把" --export

# 指定输出目录
python3 crawler.py --keyword "拖把" --export --output ./my_exports/

# 查看帮助
python3 crawler.py --help
```
