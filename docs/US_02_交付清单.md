## 🛡️ 反爬策略详解

### 1. User-Agent 轮换

- **数量**: 7 个不同的 UA
- **覆盖**: Windows/Mac/Linux + Chrome/Firefox/Safari/Edge
- **策略**: 每次请求随机选择一个 UA
- **方法**: `_get_random_ua()`, `_get_headers()`

### 2. 请求频率控制

- **默认范围**: 2-5 秒（可配置）
- **策略**: 随机延迟，避免固定模式
- **方法**: `_get_random_delay()`, `_apply_rate_limiting()`

### 3. 代理池集成

- **初始化**: `_init_proxy_pool()`
- **选择**: 随机选择可用代理
- **切换**: 失败时自动切换（最大重试 3 次）
- **方法**: `_get_next_proxy()`, `_switch_proxy()`

### 4. 验证码检测

- **检测方式**: 关键词匹配 + 状态码检测
- **处理**: 记录日志 → 切换代理 → 重试
- **日志**: 写入 `captcha_events.jsonl`
- **方法**: `_is_slider_captcha()`, `_log_captcha_event()`

### 5. 错误处理

- **网络错误**: 指数退避重试（1s, 2s, 4s...）
- **代理失败**: 达到上限后切换
- **验证码**: 立即切换代理
- **最大重试**: 3 次（可配置）

## 🔧 使用说明

### 基础用法

```bash
# 基础爬取（默认启用反爬策略）
cd /app/dev_project/001_1688_product_crawler/backend
python3 crawler.py --keyword "拖把" --limit 50

# 预览模式（只抓取前 5 条）
python3 crawler.py --keyword "收纳盒" --preview

# 自定义输出目录
python3 crawler.py --keyword "厨房用品" --output ./my_data/
```

### 反爬策略配置

```python
# 自定义延迟范围
crawler = ProductCrawler(
    keyword="拖把",
    min_delay=3.0,  # 最小延迟 3 秒
    max_delay=6.0   # 最大延迟 6 秒
)

# 禁用代理池
crawler = ProductCrawler(
    keyword="拖把",
    proxy_enabled=False
)

# 自定义重试次数
crawler = ProductCrawler(
    keyword="拖把",
    max_retries=5
)
```


## ⚠️ 注意事项

### 1. 代理源配置

当前代码中使用示例代理（`proxy1.example.com`），生产环境需要：

```python
# 替换为真实代理 API
self.proxy_pool = [
    {"http": "http://real-proxy1.com:8080"},
    {"http": "http://real-proxy2.com:8080"},
    # ...
]
```

**推荐方案**:
- 免费代理：`free-proxy-list.net`, `spys.me`
- 付费代理：更稳定，适合生产环境

